4. **胜率与期望收益计算**

   假设每笔交易独立、无累计回撤，采用逻辑回归模型线性组合多项技术特征，经sigmoid函数映射得到本次交易成功概率（胜率），再结合止盈/止损比计算期望收益。完整公式如下：

   $$
   \boxed{
   \begin{aligned}
   &z = w_0 + \sum_{i=1}^{n} w_i X_i, \\
   &p = \frac{1}{1 + e^{-z}}, \\
   &\mathbb{E}[R] = p\Bigl(\frac{\mathrm{TP1}-\mathrm{Entry}}{\mathrm{Entry}}\Bigr) + (1-p)\Bigl(\frac{\mathrm{TP2}-\mathrm{Entry}}{\mathrm{Entry}}\Bigr) + (1-p)\Bigl(\frac{\mathrm{TP3}-\mathrm{Entry}}{\mathrm{Entry}}\Bigr) - (1-p)\Bigl(\frac{\mathrm{Entry}-\mathrm{SL}}{\mathrm{Entry}}\Bigr)
   \end{aligned}
   }
   $$

   其中 $X_i$ 为可用技术特征（如RSI、MACD、ADX、EMA等），$w_i$ 为回测或专家拟合得到的权重，$w_0$ 为偏置项；$p$ 为本次交易胜率，$\mathbb{E}[R]$ 为单笔期望收益。TP1/TP2/TP3为多级止盈目标，Entry为入场价，SL为止损价。

   * 若仅有一个止盈目标，仅用TP1。
   * 若分批止盈，胜率可分段加权。

| $w_i$            | 特征权重，回测或专家标注MLE/最小二乘拟合获得                         | 历史信号与实际PnL对比拟合，优化各特征灵敏度                                       |
| $w_0$            | 偏置项（基线log-odds），通常为历史平均胜率的log-odds                  | $\bar p$为回测总体胜率                                                           |
| $z$              | 线性组合结果（log-odds）                                             | $z = w_0 + \sum w_i X_i$                                                         |
| $p$              | 本次交易胜率，$p\in(0,1)$                                            | $p=\sigma(z)=\frac{1}{1+e^{-z}}$                                                 |
| Entry            | 入场价（如“30min收盘>2540 USDT”）                                   | 实盘或回测实际入场价                                                              |
| SL               | 止损价（如2515 USDT）                                                | 按提示止损条件设定                                                                |
| TP               | 止盈价（如TP1=2565 USDT, TP2=2590 USDT）                            | 按提示止盈目标设定                                                                |
| $\mathbb{E}[R]$  | 单笔期望收益                                                         | $\mathbb{E}[R]=p\cdot R_{gain} - (1-p)\cdot R_{loss}$                           |
| $R_{gain}$       | 胜时相对收益 = $(TP-Entry)/Entry$                                   | 例：(2565-2540)/2540                                                              |
| $R_{loss}$       | 亏时相对损失 = $(Entry-SL)/Entry$                                   | 例：(2540-2515)/2540                                                              |

## 完整公式

$$
\boxed{
\begin{aligned}
&\underbrace{z}_{\text{log-odds}}
\;=\; w_0 \;+\; \sum_{i=1}^{n} w_i X_i, \\[6pt]
&\underbrace{p}_{\text{胜率}}
\;=\;\sigma(z)
\;=\;\frac{1}{1 + e^{-z}}, \\[6pt]
&\underbrace{\mathbb{E}[R]}_{\text{期望收益}}
\;=\; p \times \underbrace{\biggl(\frac{\mathrm{TP}-\mathrm{Entry}}{\mathrm{Entry}}\biggr)}_{R_{gain}}\;-
(1-p)\times \underbrace{\biggl(\frac{\mathrm{Entry}-\mathrm{SL}}{\mathrm{Entry}}\biggr)}_{R_{loss}}.
\end{aligned}
}
$$

*   **步骤**：

   1. 计算特征值 $X_i$ 并归一化到 $[0,1]$；
   2. 代入权重 $w_i$ 和偏置 $w_0$ 得到log-odds $z$；
   3. Sigmoid映射得到胜率 $p$；
   4. 结合止盈/止损比计算期望收益 $\mathbb{E}[R]$；
   5. **输出**：胜率 $p$ 和期望收益 $\mathbb{E}[R]$。
